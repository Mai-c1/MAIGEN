MAIGEN公共模块，其下仍有多个子模块
- `MAIGEN-common-core`：核心模块，包含公共类、枚举、异常、工具类等,比如MAIGEN-api、MAIGEN-analysis和MAIGEN-sandbox在任务执行的过程中，都会有一些公共的类，这些类就可以放到这个模块中。
- `MAIGEN-common-email`：包含邮箱所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
- `MAIGEN-common-mybatis`：包含http客户端所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
- `MAIGEN-common-redis`：包含redis所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
- `MAIGEN-common-rabbitmq`：包含rabbitmq所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
- `MAIGEN-common-satoken`：包含satoken所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
- `MAIGEN-common-miniIO`：包含miniIO所需要的依赖，工具类、配置类、配置文件等信息，其他模块按需引入
